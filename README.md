# basic_excercises
Ejercicios básicos de práctica del lenguaje de programación Gleam. Me interesa aprender el paradigma de programación funcional porque representa un gran reto para aprender cosas nuevas. 

Además, me llama la atención lo que ofrecen los lenguajes de programación funcionales. Lo que ofrecen es crear programas más seguros y libres de excepciones desconocidas.
